# studio-price-analysis-uae
SQL-based analysis of UAE studio apartment prices

## Project Overview
This project analyzes studio apartment listings in the UAE using SQL. 
The goal is to understand price trends, location-based differences, and market behavior.

## Dataset
- Source: UAE property listings (CSV file)
- Records: Studio apartment listings
- Key columns: price, location, bathrooms, addedOn

## Tools Used
- MySQL
- MySQL Workbench
- Excel (data cleaning)
- GitHub

## SQL Analysis
The analysis includes:
- Average price by location

- Monthly listing trends

## Conclusion
This analysis provides insights into the UAE studio apartment market and highlights how location, time, and amenities influence prices.
